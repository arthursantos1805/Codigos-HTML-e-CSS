TIPOS DE CAIXA
<span> = inline-level, agrupamento que ocupa apenas parte da width de uma tela e não pula parágrafo
<div> = box-level, agrupamento que ocupa toda a width de uma tela e pula parágrafo
<header> = agrupamento do cabeçalho, início do site
<main> = agrupamento do grupo principal do site
<footer> = agrupamento de informações por menores, como copyright, autor, etc...
<nav> = seção exclusiva para links
<section> = basicamente um conteúdo de h
<article> = meio auto explicativo, é pra artigos, tipo notícias, paginas de blog
<aside> = tipo créditos
<table> = cria tabelas
    HIERARQUIA DE TABELAS (simples)
    TABLE = tabela
        TABLE ROW = linha de tabela
            TABLE HEADER = cabeçalho de tabela
            TABLE DATA = dados de tabela

    ANATOMIA PARA TABELAS GRANDES
    TABLE
        CAPTION = legenda da tabela, aparece em cima de tudo
        THEAD
            TR, TD, TH
        TBODY
            TR, TD, TH
        TFOOT
            TR, TD, TH
        COLGROUP
            <col class= "cgrupo" span"quantidade de colunas">
    scope = col = as informações do th ficam abaixo dele
            row = as informações do th ficam logo ao lado
            colgroup e rowgroup = basicamente um plural de rows ou cols
    colspan = quantas colunas a data vai ocupar
    rowspan = quantas linhas a data vai ocupar

iframe = forma de colocar um site dentro de outro, como uma página nova

    frameborder = tamanho da borda

    width

name = basicamente o id de envio de dados, também se usa pra colocar target pro iframe

            allow-same-origin
            allow-forms

    referrerpolicy="no-referrer"

input = obtenção de dados
    text = caixa de texto
        required = campo obrigatório
        minlenght = minimo de caracteres
        maxlengt = máximo de caracteres
        size = tamanho da caixa em caracteres
        placeholder = texto dentro da caixa
    password = caixa de texto com censura
    submit = confirmador do action do form
    reset = limpar campos de input


form = tag de formulario
    autocomplete = off/on
    action = "fyle.php" = oq vai acontecer no submit

                                envio de arquivos)
             GET: infos na url (dados nada sensíveis, como buscas no google)
label = in-line conjuntivo (usar sempre label para garantir o junção de texto e input)
